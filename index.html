Defination:

In JavaScript,
"this" is a keyword that refers to the current execution context. The value of "this" is determined by how a function is called or how an object is used.
The behavior of "this" can vary in different situations:

1:Global Context:

When used outside any function, "this" refers to the global object, which is usually the window in a web browser.

Example:

    console.log(this)
    output  = window.object

2:Function Context:

Inside a regular function, "this" depends on how the function is called. It could be the global object or an object that the function belongs to.

Example:

    function add(){
    console.log(this)
    }

3:Method Context:

When a function is part of an object (a method), "this" refers to the object the method is called on.

Example:

    const student = {
    name :"Yasir",
    age : 28,
    eng:45,
    math:100,
    phy:85,
    obj : this,  //global scope
    // getAverage(){
    //   let average =  (eng + math + phy) / 3;
    //   return Number (average.toFixed(2));
    // // show error undefined
    // }
    getAverage(){
    let average =  (this.eng + this.math + this.phy) / 3;
    return Number (average.toFixed(2));
    // show the average
    }
    }
    console.log(student.getAverage());

4:Arrow Function Context:

Arrow functions don't have their own "this"; they inherit it from the surrounding context when they are created.

Example:

    const student1 = {
    username :"Yasir",
    age : 28,
    eng:45,
    math:100,
    phy:85,
    obj : this,
    getName(){
    return this.name;
    },
    getName2:()=>{
    return this.username;
    },
    printThis(){
    setTimeout(()=>{
    console.log(this)
    },1000)
    },
    printThis(){
    setTimeout(function(){
    console.log(this)
    },1000)
    },
    }
    console.log(student1.getName());
    console.log(student1.getName2());
    console.log(student1.printThis());

5:Event Handler Context:

In event handlers, "this" often points to the DOM element that triggered the event.

Example:

    const btn = document.querySelector("#btn");
    // btn.addEventListener("click",()=>{
    //     console.log(this)
    //     // print window object
    // })
    btn.addEventListener("click",function(){
    console.log(this)
    // print button object
    })
